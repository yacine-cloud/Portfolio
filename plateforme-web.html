ECE In – Plateforme de réseau social professionnel

ECE – ING3 Web dynamique · 2024

Objectif du projet

Concevoir et développer ECE In, une plateforme web de type réseau social professionnel destinée à la communauté ECE Paris (étudiants, enseignants, alumni, personnels). Les utilisateurs peuvent créer leur profil, gérer leur réseau, publier des contenus et accéder à des opportunités d’emploi.

Contexte

Projet « piscine » de Web dynamique ING3 réalisé en trinôme/quadrinôme. L’application suit une architecture client–serveur avec base MySQL, back‑end en PHP et front‑end en HTML / CSS / JavaScript.

Le sujet impose de se rapprocher d’un LinkedIn interne à l’ECE : profils, réseau, messagerie, publications, emplois, administration et gestion des droits.

Fonctionnalités principales

Pages & navigation

  • Accueil : présentation, événement de la semaine, carrousel, flux du réseau.
  • Mon Réseau : contacts, suggestions, profils liés.
  • Vous : profil complet, expériences, projets, CV généré automatiquement.
  • Notifications : événements, annonces, nouveautés du réseau.
  • Messagerie : chat texte avec historique.
  • Emplois : offres CDI, CDD, stages, alternances.

Social & contenu

  • Publication d’événements, photos, vidéos, CV.
  • Paramètres de visibilité (public / amis / personnalisé).
  • Réactions : like, commentaire, partage.
  • Albums photos.
  • Gestion des amis et recommandations.
  • Thèmes graphiques selon événements (Noël, fêtes…).

Gestion des utilisateurs & sécurité

  • Deux types de comptes : auteur et administrateur.
  • Création de compte par admin (adresse ECE, pseudo, nom).
  • Authentification sécurisée, sessions, pages protégées.
  • Interface admin : ajout/suppression de comptes.
  • Gestion des droits selon le type d’utilisateur et la confidentialité.

Architecture technique

Front-end

  • HTML structuré (header/nav, contenu, footer).
  • CSS pour layout, responsivité, thèmes.
  • JavaScript pour interactions (menus, carrousel, dynamique).
  • Possibilité d’utiliser Bootstrap.

Back-end & base de données

  • Back-end en PHP : logique métier, formulaires, sessions.
  • Base MySQL : utilisateurs, messages, posts, événements, offres.
  • Modèle entité‑association → modèle relationnel.
  • Scripts SQL d’import/export.

Rôle personnel & développement

  • Développement full‑stack : HTML/CSS, JS, PHP, MySQL.
  • Implémentation d’un système d’authentification sécurisé.
  • Conception de la base et requêtes CRUD.
  • Développement du feed, interactions, réseau, profils.
  • Versioning GitHub et gestion de projet.
Dépôt du projet : github.com/yacine-cloud/projet-web

Résultats

La plateforme fournit un réseau social professionnel complet pour les membres de l’ECE : profils, réseau, publications, messagerie, offres d’emploi et administration. L’ensemble respecte les contraintes du sujet (fonctionnalités, technologies, deadlines).
