# Portfolio — Projets d’ingénierie & développement

Bienvenue dans mon **portfolio technique**, qui regroupe une sélection de projets réalisés en école d’ingénieur (ECE Paris) et en autodidacte.  
Il met en avant mes compétences en :

- Développement embarqué (microcontrôleurs, capteurs, GLCD, USB…)
- Développement web full‑stack (HTML / CSS / JS / PHP / MySQL)
- Électronique analogique & numérique (Arduino, traitement du signal)
- Programmation en C, algorithmique, logique de jeu

Le site est entièrement développé en **HTML / CSS / JavaScript** (sans framework) afin de rester léger, simple et facilement hébergeable.

---

##  Contenu du portfolio

Chaque projet dispose d’une page dédiée comprenant :

- Description technique  
- Objectifs et contexte  
- Fonctionnalités clés  
- Architecture / schémas  
- Compétences mobilisées  
- Captures d’écran  
- Lien vers le code source (si disponible)

### 🎮 Jeu embarqué “FlapPIC Bird”
- Développement en C + assembleur sur PIC18  
- Affichage GLCD, gestion capteurs et interruptions  
- Communication USB ↔ Python pour contrôle en temps réel

### 🌐 Plateforme web “ECE In”
- Réseau social interne pour étudiants

### Radio Arduino — Modulation & démodulation

###  Jeu Quoridor (C – console)

- IA simple, gestion du plateau et des règles  
- Sauvegarde / chargement de partie
