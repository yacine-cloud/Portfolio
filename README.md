## Portfolio — Projets d’ingénierie & développement

Bienvenue dans mon **portfolio technique**, regroupant une sélection de projets réalisés en école d’ingénieur (ECE Paris) et en autodidacte.  
Ce site présente des travaux concrets en :

- Développement embarqué (microcontrôleurs, capteurs, GLCD, USB…)
- Développement web full-stack (HTML / CSS / JS / PHP / MySQL)
- Électronique analogique et numérique (Arduino, traitement du signal)
- Programmation en C, algorithmique, logique de jeu

Ce portfolio est entièrement codé en **HTML / CSS / JavaScript** (sans framework) pour rester simple, léger et facilement hébergeable.

---

## Contenu

Chaque projet dispose d’une page dédiée avec :

- Description technique  
- Objectifs et contexte  
- Fonctionnalités clés  
- Architecture et schémas  
- Compétences mobilisées  
- Captures d’écran  
- Lien vers le code source (si disponible)

## Projets présentés

### 🎮 Jeu embarqué “FlapPIC Bird”
- Développement en C + assembleur sur PIC18  
- Affichage GLCD, gestion capteurs, interruptions  
- Communication USB ↔ Python pour contrôle en temps réel

### 🌐 Plateforme web “ECE In”
- Réseau social interne pour étudiants  
- Stack PHP / MySQL / JavaScript  
- Authentification, posts, feed, interface responsive

### 📡 Radio Arduino — Modulation & démodulation
- Conception d’un circuit analogique complet  
- Transmission audio via modulation simple  
- Composants : diodes, transistors, haut-parleur, microphone

### ♟️ Jeu Quoridor (C – console)
- Jeu combinatoire 2 à 4 joueurs  
- IA simple, gestion du plateau et des règles  
- Sauvegarde / chargement de partie
