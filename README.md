Portfolio – Projets d’ingénieur & développement

Bienvenue dans mon **portfolio personnel**, un site web regroupant l’ensemble de mes projets réalisés en école d’ingénieur (ECE Paris), au lycée, et en autodidacte.  
Il présente mes travaux en :

- Développement embarqué (microcontrôleurs, capteurs, GLCD, USB…)
- Développement web full-stack (HTML / CSS / JS / PHP / MySQL)
- Électronique / Arduino / traitement du signal
- Programmation en C, algorithmique & logique de jeu
- 
Ce portfolio est entièrement codé en **HTML / CSS / JavaScript** (sans framework) pour rester simple, léger et facile à héberger.

---

## Contenu du portfolio

Chaque projet possède sa propre page dédiée avec :

- Description complète  
- Objectifs  
- Fonctionnalités  
- Architecture  
- Compétences développées  
- Images / captures  
- Liens vers le code source lorsque disponible  

### Projets inclus :

#### Microcontrôleur – Jeu “FlapPIC Bird”
- Développement en C + assembleur sur PIC18  
- Affichage sur GLCD  
- Gestion capteurs, interruptions, USB ↔ Python  
- Mini-jeu inspiré de Flappy Bird

#### Projet Web – ECE In
- Plateforme de réseau social interne à l’ECE  
- PHP / MySQL / JavaScript  
- Authentification, posts, amis, admin, feed  
- Carrousel automatique + interface responsive

#### Radio Arduino – Modulation & démodulation
- Circuit électronique complet (diodes, transistors…)  
- Transmission d’un signal audio  
- Reproduction d’un système radio élémentaire

#### Jeu Quoridor (C – console)
- Jeu combinatoire 2 à 4 joueurs  
- IA simple  
- Gestion du plateau, des murs, déplacements  
- Sauvegarde / chargement de partie
